Ejemplo de manejo de GIT



  • Inicializamos el repositorio con git init.
  • Verificar si hay una conexión remota con el comando git remote -v.
  • Revisamos acciones que estén pendientes por efectuarse con el comando git status
  • Adicionamos archivos al repositorio con git add "NombreDelArchivo.extencion"
  • Para adicionar texto de apoyo al archivo que esta en el repositorio Se utiliza git commit -m "Texto".
  • git push: Para garantizar que se allá almacenado en la nube los archivos.
  • git add . : Para adicionar todos los archivos al repositorio.
  • clear: Limpiar la consola.
